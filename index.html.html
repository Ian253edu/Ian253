Esta es una página de demostración con HTML, CSS y JavaScript separados.

HTML5

HTML (HyperText Markup Language) es el lenguaje estándar para crear páginas web.

  • Estructura semántica
  • Formularios y botones
  • Elementos multimedia
  • Accesibilidad básica
  • Meta etiquetas para responsive design
CSS3

CSS (Cascading Style Sheets) se utiliza para dar estilo a documentos HTML.

  • Diseño responsive
  • Transiciones y animaciones
  • Variables CSS personalizadas
  • Efectos de sombra y gradientes
  • Diseño de tarjetas modernas
JavaScript

JavaScript añade interactividad a las páginas web.

  • Manipulación del DOM
  • Event listeners
  • Animaciones dinámicas
  • Generación de contenido
  • Manipulación de estilos CSS
